Cloud and Skydome texture(s)
By Cr4zy

Created for use in my Sky Shader tutorial for UT3/UDK that can be found at http://cr4zyb4st4rd.co.uk/tutorial.html


=====================================================

General Info
_____________________________________________________

.TGA fileformat

x8 - 1024x1024 Tiling B&W Cloud textures
x2 -  256x256  Tiling Distorting Patterns
x1 - 1024x1024 Skydome texture
x1 -  256x256  Sky Mask

.UPK fileformat

x1 - Package for Unreal Editor with textures imported and material setup - Will work in UT3 and UDK
